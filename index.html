Explain Var, Let and Const

    Var:

  • Var is a functional as well as Global Scope
  • Var can be reinitialized and redeclared
  • var can be modified later

    Let:

  • Let is a Block Scope
  • Let cannot be reinitialized and redeclared
  • Let can be modified later

    Const:

  • Const is a Block Scope
  • Const cannot be reinitialized and redeclared
  • Const cannot be modified later

What is Hoisting?

Hoising is a phenemenon by which we can access vaiables and functions even before we initialize it.In case of var it is undefined but in case of let and const it throws a Reference Error.

What is Temporal Dead Zone?

It is the time between the hoisting and the initialization of variables. Even let and const are hoisted but they are in different space called Temporal Dead Zone.
